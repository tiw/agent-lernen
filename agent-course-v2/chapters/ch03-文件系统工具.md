## 🧠 核心概念

### 为什么需要专门的文件工具？

第 2 章的 BashTool 虽然也能读写文件（`cat file.txt`、`echo "xxx" > file.txt`），但有很多问题：

| 方式 | 优点 | 缺点 |
|------|------|------|
| **Bash 读写** | 简单直接 | 不安全、无权限控制、输出格式混乱 |
| **专用文件工具** | 安全沙箱、结构化输出、精准编辑 | 需要额外实现 |

专用文件工具的优势：
- 🔒 **安全沙箱**：限制 Agent 只能访问指定目录
- 📄 **结构化输出**：带行号、带元信息，LLM 更容易理解
- ✏️ **精准编辑**：SEARCH/REPLACE 模式，只改需要改的地方
- 📊 **Token 预算**：自动截断大文件，防止撑爆上下文

### 安全沙箱（Sandbox）

沙箱是文件工具的安全边界。想象一个透明的玻璃箱——Agent 只能看到和操作箱子里的东西：

```
┌─────────────────────────────────────┐
│           安全沙箱                   │
│                                     │
│  ✅ /home/user/project/             │
│     ├── src/                        │
│     ├── tests/                      │
│     └── README.md                   │
│                                     │
│  ❌ /etc/passwd          ← 越权！    │
│  ❌ /root/.ssh/          ← 越权！    │
│  ❌ /dev/random          ← 越权！    │
│                                     │
└─────────────────────────────────────┘
```

**沙箱的核心职责**：
1. **路径解析**：把 `../` 等相对路径解析为绝对路径
2. **权限校验**：检查路径是否在允许的目录树内
3. **设备拦截**：阻止访问 `/dev/zero`、`/dev/random` 等设备文件
4. **路径穿越防护**：防止 `../../../etc/passwd` 这类攻击

### 三种文件操作

| 工具 | 用途 | 适用场景 |
|------|------|----------|
| **FileReadTool** | 读取文件内容 | 查看代码、配置文件、日志 |
| **FileWriteTool** | 创建或覆盖文件 | 新建文件、大改动 |
| **FileEditTool** | 精准替换（SEARCH/REPLACE） | 小改动、变量重命名 |

**编辑策略建议**：
- 新文件 → `FileWriteTool`
- 小改动（几行） → `FileEditTool`
- 大改动（整个文件重写） → `FileWriteTool`

class SandboxViolationError(Exception):
    """路径超出沙箱范围"""
    pass


# ============================================================
# 安全沙箱
# ============================================================

class FileSandbox:
    """
    安全沙箱：限制文件操作只能在允许的目录内进行。
    
    使用示例：
        sandbox = FileSandbox(allowed_dirs=["/home/user/project"])
        path = sandbox.validate_path("src/main.py")  # ✅
        path = sandbox.validate_path("/etc/passwd")   # ❌ SandboxViolationError
    """
    
    # 禁止访问的设备文件
    BLOCKED_DEVICES = frozenset({
        '/dev/zero', '/dev/random', '/dev/urandom',
        '/dev/full', '/dev/stdin', '/dev/stdout',
        '/dev/stderr', '/dev/null', '/dev/tty',
    })
    
    def __init__(self, allowed_dirs: Optional[list[str]] = None):
        """
        Args:
            allowed_dirs: 允许访问的目录列表。
                         如果为空或 None，则不限制（⚠️ 仅用于测试）。
        """
        self.allowed_dirs = [
            Path(d).resolve() for d in (allowed_dirs or [])
        ]
    
    def validate_path(self, path: str | Path) -> Path:
        """
        校验路径是否在沙箱范围内。
        
        安全检查步骤：
        1. 解析为绝对路径（处理 ../ 等相对路径）
        2. 检查是否为设备文件
        3. 检查是否在允许的目录树内
        
        Args:
            path: 文件路径（绝对或相对）
            
        Returns:
            解析后的绝对路径
            
        Raises:
            SandboxViolationError: 路径超出沙箱范围
        """
        # 1. 解析为绝对路径（关键！处理 ../ 穿越）
        resolved = Path(path).resolve()
        
        # 2. 检查设备文件
        if str(resolved) in self.BLOCKED_DEVICES:
            raise SandboxViolationError(
                f"Cannot access device file: {resolved}"
            )
        
        # 3. 如果没设置沙箱，不限制
        if not self.allowed_dirs:
            return resolved
        
        # 4. 检查是否在允许的目录树内
        for allowed in self.allowed_dirs:
            try:
                resolved.relative_to(allowed)
                return resolved  # ✅ 在允许的目录内
            except ValueError:
                continue  # 不在这个目录树内，检查下一个
        
        # ❌ 所有允许的目录都不匹配
        raise SandboxViolationError(
            f"Path '{resolved}' is outside allowed directories: "
            f"{[str(d) for d in self.allowed_dirs]}"
        )

@dataclass
class FileReadResult:
    """文件读取结果"""
    file_path: str
    content: str
    num_lines: int
    start_line: int
    total_lines: int
    encoding: str = 'utf-8'
    
    def to_display(self) -> str:
        """格式化为带行号的显示内容（类似 cat -n）"""
        lines = self.content.split('\n')
        numbered = []
        for i, line in enumerate(lines, start=self.start_line):
            numbered.append(f"{i:>6}\t{line}")
        return '\n'.join(numbered)


@dataclass
class FileWriteResult:
    """文件写入结果"""
    file_path: str
    operation: str  # 'create' 或 'update'
    bytes_written: int
    diff: str = ''


@dataclass
class FileEditResult:
    """文件编辑结果"""
    file_path: str
    old_string: str
    new_string: str
    success: bool
    message: str
    diff: str = ''
    occurrences: int = 0


# ============================================================
# FileReadTool
# ============================================================

class FileReadTool:
    """
    读取文件内容。
    
    特性：
    - 分页读取（offset / limit）
    - 带行号输出
    - Token 估算和限制
    - 安全沙箱校验
    
    使用示例：
        tool = FileReadTool(sandbox=my_sandbox)
        result = tool.call("src/main.py", offset=1, limit=50)
        print(result.to_display())
    """
    
    def __init__(
        self,
        sandbox: Optional[FileSandbox] = None,
        max_lines: int = 2000,
        max_tokens: int = 25000,
    ):
        """
        Args:
            sandbox: 安全沙箱实例
            max_lines: 单次读取最大行数
            max_tokens: 单次读取最大 token 估算值
        """
        self.sandbox = sandbox or FileSandbox()
        self.max_lines = max_lines
        self.max_tokens = max_tokens
    
    @property
    def name(self) -> str:
        return "file_read"
    
    @property
    def description(self) -> str:
        return "读取文件内容。支持指定起始行号和读取行数。"
    
    @property
    def parameters(self) -> dict:
        return {
            "type": "object",
            "properties": {
                "file_path": {
                    "type": "string",
                    "description": "文件路径（绝对或相对）"
                },
                "offset": {
                    "type": "integer",
                    "description": "起始行号（1-indexed），默认 1"
                },
                "limit": {
                    "type": "integer",
                    "description": "最多读取的行数，默认 2000"
                }
            },
            "required": ["file_path"],
        }
    
    def call(
        self,
        file_path: str,
        offset: int = 1,
        limit: Optional[int] = None,
        encoding: str = 'utf-8',
    ) -> FileReadResult:
        """
        读取文件内容。
        
        Args:
            file_path: 文件路径
            offset: 起始行号（1-indexed）
            limit: 最多读取的行数
            encoding: 文件编码
            
        Returns:
            FileReadResult 包含文件内容和元信息
        """
        # 1. 安全校验
        resolved = self.sandbox.validate_path(file_path)
        
        if not resolved.exists():
            raise FileNotFoundError(
                f"File not found: {resolved}\n"
                f"Current working directory: {os.getcwd()}"
            )
        
        if not resolved.is_file():
            raise IsADirectoryError(
                f"Path is a directory, not a file: {resolved}"
            )
        
        # 2. 读取全部行
        with open(resolved, 'r', encoding=encoding) as f:
            all_lines = f.readlines()
        
        total_lines = len(all_lines)
        
        # 3. 计算读取范围
        start = max(0, offset - 1)  # 转为 0-indexed
        end = start + (limit or self.max_lines)
        end = min(end, total_lines)
        
        selected_lines = all_lines[start:end]
        content = ''.join(selected_lines)
        
        # 4. Token 估算检查（粗略估算：1 token ≈ 4 个英文字符 / 1.5 个中文字符）
        estimated_tokens = len(content) // 4
        if estimated_tokens > self.max_tokens:
            raise ValueError(
                f"File content ({estimated_tokens} estimated tokens) exceeds "
                f"maximum ({self.max_tokens} tokens). "
                f"Use offset and limit to read a smaller portion."
            )
        
        return FileReadResult(
            file_path=str(resolved),
            content=content,
            num_lines=end - start,
            start_line=offset,
            total_lines=total_lines,
            encoding=encoding,
        )
    
    def execute(self, **kwargs) -> str:
        """Tool 基类接口：执行并返回字符串结果"""
        result = self.call(
            file_path=kwargs["file_path"],
            offset=kwargs.get("offset", 1),
            limit=kwargs.get("limit"),
            encoding=kwargs.get("encoding", "utf-8"),
        )
        
        # 添加元信息头
        header = (
            f"📄 {result.file_path} "
            f"(第 {result.start_line}-{result.start_line + result.num_lines - 1} 行，"
            f"共 {result.total_lines} 行)\n"
        )
        return header + result.to_display()

class FileWriteTool:
    """
    写入文件（创建或覆盖）。
    
    特性：
    - 自动创建父目录
    - 区分创建（create）和更新（update）
    - 生成 diff 展示变更
    
    使用示例：
        tool = FileWriteTool(sandbox=my_sandbox)
        result = tool.call("src/new_file.py", content="print('hello')")
    """
    
    def __init__(self, sandbox: Optional[FileSandbox] = None):
        self.sandbox = sandbox or FileSandbox()
    
    @property
    def name(self) -> str:
        return "file_write"
    
    @property
    def description(self) -> str:
        return "创建或覆盖文件内容。如果文件不存在会自动创建。"
    
    @property
    def parameters(self) -> dict:
        return {
            "type": "object",
            "properties": {
                "file_path": {
                    "type": "string",
                    "description": "目标文件路径"
                },
                "content": {
                    "type": "string",
                    "description": "要写入的文件内容"
                }
            },
            "required": ["file_path", "content"],
        }
    
    def call(
        self,
        file_path: str,
        content: str,
        encoding: str = 'utf-8',
    ) -> FileWriteResult:
        """
        写入文件内容。
        
        Args:
            file_path: 目标文件路径
            content: 要写入的内容
            encoding: 文件编码
            
        Returns:
            FileWriteResult 包含操作类型和 diff
        """
        resolved = self.sandbox.validate_path(file_path)
        
        # 检测是创建还是更新
        operation = 'update' if resolved.exists() else 'create'
        old_content = ''
        if operation == 'update':
            with open(resolved, 'r', encoding=encoding) as f:
                old_content = f.read()
        
        # 自动创建父目录
        resolved.parent.mkdir(parents=True, exist_ok=True)
        
        # 写入文件
        with open(resolved, 'w', encoding=encoding) as f:
            f.write(content)
        
        # 生成 diff
        diff = self._make_diff(old_content, content, str(resolved))
        
        return FileWriteResult(
            file_path=str(resolved),
            operation=operation,
            bytes_written=len(content.encode(encoding)),
            diff=diff,
        )
    
    def execute(self, **kwargs) -> str:
        """Tool 基类接口"""
        result = self.call(
            file_path=kwargs["file_path"],
            content=kwargs["content"],
            encoding=kwargs.get("encoding", "utf-8"),
        )
        
        op_emoji = "✅ 创建" if result.operation == "create" else "🔄 更新"
        return (
            f"{op_emoji} 文件: {result.file_path}\n"
            f"写入 {result.bytes_written} 字节\n"
            f"{result.diff}"
        )
    
    @staticmethod
    def _make_diff(old: str, new: str, path: str) -> str:
        """生成 unified diff"""
        if not old:
            return f"New file created: {path}"
        
        diff_lines = difflib.unified_diff(
            old.splitlines(keepends=True),
            new.splitlines(keepends=True),
            fromfile=f"a/{path}",
            tofile=f"b/{path}",
        )
        return ''.join(diff_lines)

class FileEditTool:
    """
    精准编辑文件（SEARCH/REPLACE 模式）。
    
    特性：
    - old_string / new_string 精确匹配替换
    - replace_all 支持全局替换
    - 字符串未找到时提供友好错误信息
    - 生成 diff 展示变更
    
    使用示例：
        tool = FileEditTool(sandbox=my_sandbox)
        result = tool.call(
            "src/main.py",
            old_string="print('hello')",
            new_string="print('hello, world!')"
        )
    """
    
    def __init__(self, sandbox: Optional[FileSandbox] = None):
        self.sandbox = sandbox or FileSandbox()
    
    @property
    def name(self) -> str:
        return "file_edit"
    
    @property
    def description(self) -> str:
        return (
            "精准编辑文件内容。使用 SEARCH/REPLACE 模式替换指定字符串。"
            "old_string 必须在文件中唯一匹配，否则需要设置 replace_all=True。"
        )
    
    @property
    def parameters(self) -> dict:
        return {
            "type": "object",
            "properties": {
                "file_path": {
                    "type": "string",
                    "description": "目标文件路径"
                },
                "old_string": {
                    "type": "string",
                    "description": "要替换的原文（必须精确匹配）"
                },
                "new_string": {
                    "type": "string",
                    "description": "替换后的新内容"
                },
                "replace_all": {
                    "type": "boolean",
                    "description": "是否替换所有匹配项，默认 False"
                }
            },
            "required": ["file_path", "old_string", "new_string"],
        }
    
    def call(
        self,
        file_path: str,
        old_string: str,
        new_string: str,
        replace_all: bool = False,
        encoding: str = 'utf-8',
    ) -> FileEditResult:
        """
        在文件中执行精确的字符串替换。
        
        Args:
            file_path: 目标文件路径
            old_string: 要替换的原文（必须唯一匹配）
            new_string: 替换后的新内容
            replace_all: 是否替换所有匹配项
            encoding: 文件编码
            
        Returns:
            FileEditResult 包含编辑结果
        """
        resolved = self.sandbox.validate_path(file_path)
        
        # 1. 无变化检查
        if old_string == new_string:
            return FileEditResult(
                file_path=str(resolved),
                old_string=old_string,
                new_string=new_string,
                success=False,
                message="No changes to make: old_string and new_string are identical.",
            )
        
        # 2. 文件存在性检查
        if not resolved.exists():
            return FileEditResult(
                file_path=str(resolved),
                old_string=old_string,
                new_string=new_string,
                success=False,
                message=f"File does not exist: {resolved}",
            )
        
        # 3. 读取文件
        with open(resolved, 'r', encoding=encoding) as f:
            content = f.read()
        
        # 4. 查找匹配
        occurrences = content.count(old_string)
        
        if occurrences == 0:
            return FileEditResult(
                file_path=str(resolved),
                old_string=old_string,
                new_string=new_string,
                success=False,
                message=(
                    f"String to replace not found in file.\n"
                    f"String: {old_string!r}\n\n"
                    f"💡 提示：确保 old_string 完全匹配，包括空格和换行。"
                ),
            )
        
        if occurrences > 1 and not replace_all:
            return FileEditResult(
                file_path=str(resolved),
                old_string=old_string,
                new_string=new_string,
                success=False,
                message=(
                    f"Found {occurrences} matches of the string to replace, "
                    f"but replace_all is False.\n"
                    f"💡 解决方案：\n"
                    f"  1. 设置 replace_all=True 替换所有匹配项\n"
                    f"  2. 提供更多上下文使 old_string 唯一\n"
                    f"String: {old_string!r}"
                ),
            )
        
        # 5. 执行替换
        if replace_all:
            new_content = content.replace(old_string, new_string)
        else:
            new_content = content.replace(old_string, new_string, 1)
        
        # 6. 写回文件
        with open(resolved, 'w', encoding=encoding) as f:
            f.write(new_content)
        
        # 7. 生成 diff
        diff = self._make_diff(content, new_content, str(resolved))
        
        return FileEditResult(
            file_path=str(resolved),
            old_string=old_string,
            new_string=new_string,
            success=True,
            message=f"The file {resolved} has been updated successfully.",
            diff=diff,
            occurrences=occurrences,
        )
    
    def execute(self, **kwargs) -> str:
        """Tool 基类接口"""
        result = self.call(
            file_path=kwargs["file_path"],
            old_string=kwargs["old_string"],
            new_string=kwargs["new_string"],
            replace_all=kwargs.get("replace_all", False),
            encoding=kwargs.get("encoding", "utf-8"),
        )
        
        if result.success:
            emoji = "✅" if result.occurrences == 1 else "🔄"
            return (
                f"{emoji} 编辑成功: {result.file_path}\n"
                f"替换了 {result.occurrences} 处\n"
                f"{result.diff}"
            )
        else:
            return f"❌ 编辑失败: {result.message}"
    
    @staticmethod
    def _make_diff(old: str, new: str, path: str) -> str:
        """生成 unified diff"""
        diff_lines = difflib.unified_diff(
            old.splitlines(keepends=True),
            new.splitlines(keepends=True),
            fromfile=f"a/{path}",
            tofile=f"b/{path}",
        )
        return ''.join(diff_lines)

# 便捷工厂函数
# ============================================================

def create_file_tools(
    allowed_dirs: Optional[list[str]] = None,
    max_lines: int = 2000,
    max_tokens: int = 25000,
) -> tuple[FileReadTool, FileWriteTool, FileEditTool]:
    """
    创建一组文件工具，共享同一个沙箱配置。
    
    Args:
        allowed_dirs: 允许访问的目录列表
        max_lines: 单次读取最大行数
        max_tokens: 单次读取最大 token 估算值
        
    Returns:
        (read_tool, write_tool, edit_tool)
    
    使用示例：
        read_tool, write_tool, edit_tool = create_file_tools(
            allowed_dirs=["/home/user/project"]
        )
        agent = Agent(tools=[read_tool, write_tool, edit_tool])
    """
    sandbox = FileSandbox(allowed_dirs)
    return (
        FileReadTool(sandbox=sandbox, max_lines=max_lines, max_tokens=max_tokens),
        FileWriteTool(sandbox=sandbox),
        FileEditTool(sandbox=sandbox),
    )

## ⚠️ 常见错误

### 错误 1：`SandboxViolationError: Path is outside allowed directories`
**原因**：文件路径不在沙箱允许的目录内。
**解决**：
```python
# 检查沙箱配置
sandbox = FileSandbox(allowed_dirs=["/your/project/path"])
# 或者在测试时不设置沙箱
sandbox = FileSandbox()  # 不限制
```

### 错误 2：`FileEditTool 编辑失败：String to replace not found`
**原因**：`old_string` 与文件内容不完全匹配（空格、换行、引号差异）。
**解决**：
- 先用 `FileReadTool` 读取文件，确认原文内容
- 复制粘贴原文作为 `old_string`，不要手动输入
- 注意缩进（空格 vs Tab）

### 错误 3：`ValueError: File content exceeds maximum tokens`
**原因**：文件太大，超过了 token 限制。
**解决**：
```python
# 方法 1：使用 offset 和 limit 分段读取
result = tool.call("large_file.py", offset=1, limit=500)

# 方法 2：增大限制（谨慎！）
tool = FileReadTool(max_tokens=50000)
```

### 错误 4：`FileEditTool: Found N matches but replace_all is False`
**原因**：`old_string` 在文件中出现多次，不确定替换哪一个。
**解决**：
- 提供更多上下文使 `old_string` 唯一（包含前后几行）
- 或者设置 `replace_all=True` 替换所有匹配项

## 📝 本章小结

本章实现了 AI Agent 的三个核心文件工具：

| 工具 | 作用 | 关键特性 |
|------|------|----------|
| **FileReadTool** | 读取文件 | 分页读取、行号显示、Token 限制 |
| **FileWriteTool** | 创建/覆盖文件 | 自动创建目录、diff 生成 |
| **FileEditTool** | 精准编辑 | SEARCH/REPLACE、全局替换、匹配校验 |

关键设计原则：
1. 🔒 **安全沙箱**：限制可访问目录，防止 Agent 误操作系统文件
2. 📊 **Token 预算**：自动截断大文件，防止撑爆上下文窗口
3. ✏️ **差异优先**：Edit 工具只传输变更部分
4. 📝 **友好错误**：匹配失败时给出具体建议

### 当前局限性

- ❌ 文件编码固定为 UTF-8，不支持自动检测
- ❌ 没有 undo/redo 功能
- ❌ 不支持多编辑块（一次调用多个 SEARCH/REPLACE）
- ❌ 沙箱只限制目录，不限制文件类型

def detect_encoding(file_path: Path) -> str:
    """自动检测文件编码"""
    # 方法 1：尝试常见编码
    for enc in ['utf-8', 'gbk', 'latin-1', 'utf-16']:
        try:
            with open(file_path, 'r', encoding=enc) as f:
                f.read()
            return enc
        except (UnicodeDecodeError, UnicodeError):
            continue
    return 'utf-8'  # 默认回退

### 练习 2：实现文件编辑的 undo 功能
为 `FileEditTool` 添加 undo 能力。

**参考答案**：
```python
class FileEditTool:
    def __init__(self, *args, **kwargs):
        # ... 原有初始化
        self._undo_stack: dict[str, list[str]] = {}  # file_path → [old_content, ...]
    
    def call(self, file_path, old_string, new_string, ...):
        # 在执行替换前，保存当前内容
        resolved = self.sandbox.validate_path(file_path)
        if resolved.exists():
            with open(resolved, 'r') as f:
                content = f.read()
            self._undo_stack.setdefault(str(resolved), []).append(content)
        
        # ... 原有替换逻辑
    
    def undo(self, file_path: str) -> str:
        """撤销上一次编辑"""
        resolved = self.sandbox.validate_path(file_path)
        stack = self._undo_stack.get(str(resolved), [])
        if not stack:
            return "Nothing to undo."
        prev_content = stack.pop()
        with open(resolved, 'w') as f:
            f.write(prev_content)
        return f"Undo successful: {resolved}"
```
