# 第 10 章：多智能体协作 —— Team Mode

> 一个 Agent 再聪明也有极限。当任务复杂到需要分工协作时，就需要**多智能体系统**——让不同的 Agent 扮演不同角色，各司其职，协同完成目标。

---

## 🔍 先看 Claude Code 怎么做

Claude Code 的多智能体协作功能（Agent Swarms / Team Mode）是整个平台最激动人心的特性之一。它让一个 Agent 可以创建和管理多个子 Agent，组成一个"团队"来完成复杂任务。

```typescript
async call(input, context) {
  // 1. 检查是否已经在团队中（一个 leader 只能管理一个团队）
  const existingTeam = appState.teamContext?.teamName
  if (existingTeam) {
    throw new Error('Already leading a team...')
  }

  // 2. 生成唯一的团队名称
  const finalTeamName = generateUniqueTeamName(team_name)

  // 3. 为团队 leader 生成确定性的 Agent ID
  const leadAgentId = formatAgentId(TEAM_LEAD_NAME, finalTeamName)

  // 4. 创建 TeamFile（团队配置文件）
  const teamFile: TeamFile = {
    name: finalTeamName,
    description,
    createdAt: Date.now(),
    leadAgentId,
    leadSessionId: getSessionId(),
    members: [{
      agentId: leadAgentId,
      name: TEAM_LEAD_NAME,
      agentType: leadAgentType,
      model: leadModel,
      joinedAt: Date.now(),
      cwd: getCwd(),
      subscriptions: [],
    }],
  }

  // 5. 写入团队配置文件
  await writeTeamFileAsync(finalTeamName, teamFile)

  // 6. 创建对应的任务列表目录
  await resetTaskList(sanitizeName(finalTeamName))
  await ensureTasksDir(taskListId)

  // 7. 更新 AppState 中的团队上下文
  setAppState(prev => ({
    ...prev,
    teamContext: {
      teamName: finalTeamName,
      teamFilePath,
      leadAgentId,
      teammates: { ... },
    },
  }))
}
```

### 角色设计

| 角色 | 职责 | 能力 |
|------|------|------|
| **Planner（规划者）** | 分解任务、制定计划、分配角色 | 分析、推理、规划 |
| **Researcher（研究者）** | 收集信息、查找资料、整理知识 | 搜索、阅读、总结 |
| **Coder（编码者）** | 编写代码、实现功能 | 编程、调试 |
| **Reviewer（审查者）** | 代码审查、质量检查 | 分析、批判性思维 |
| **Tester（测试者）** | 编写测试、验证功能 | 测试、验证 |
| **Writer（写作者）** | 撰写文档、生成报告 | 写作、表达 |

### 通信模式

```
1. 直接消息（Direct Message）
   Lead ──"请研究 X"──► Researcher
   Researcher ──"研究完成"──► Lead

2. 任务板（Task Board）
   Lead 在任务板上创建任务
   空闲 Agent 自动认领任务
   完成后更新任务状态

3. 广播（Broadcast）
   Lead ──"所有人注意：需求变更"──► [All Agents]
```

class BaseAgent(ABC):
    """
    Agent 基类（参考 Claude Code 的 Agent 概念）

    每个 Agent 实例代表一个角色，可以：
    - 接收任务
    - 执行任务
    - 返回结果
    - 发送消息
    """

    def __init__(self, role: AgentRole):
        self.role = role
        self.current_task: Optional[str] = None
        self.is_busy = False
        self._message_queue: asyncio.Queue = asyncio.Queue()

    @abstractmethod
    async def execute(self, task_description: str, context: dict = None) -> str:
        """
        执行任务

        Args:
            task_description: 任务描述
            context: 上下文信息（如研究成果、代码等）

        Returns:
            执行结果
        """
        ...

    async def receive_message(self, message: str) -> None:
        """接收消息"""
        await self._message_queue.put(message)

    async def get_message(self) -> Optional[str]:
        """获取消息"""
        try:
            return self._message_queue.get_nowait()
        except asyncio.QueueEmpty:
            return None

    @property
    def is_available(self) -> bool:
        return not self.is_busy

    def __repr__(self) -> str:
        status = "busy" if self.is_busy else "idle"
        return f"Agent({self.role.name}, {status})"

### 2. 消息总线 `team/message_bus.py`

```python
"""
team/message_bus.py —— 多智能体消息总线

参考 Claude Code 的：
- enqueuePendingNotification（通知队列）
- SendMessage tool（Agent 间通信）
- team member subscriptions
"""

import asyncio
import logging
import time
from dataclasses import dataclass, field
from typing import Optional

logger = logging.getLogger(__name__)


@dataclass
class Message:
    """Agent 间消息"""
    sender: str
    receiver: str
    content: str
    timestamp: float = field(default_factory=time.time)
    message_type: str = "direct"  # "direct" | "broadcast" | "system"

    def __repr__(self) -> str:
        return f"Message({self.sender} → {self.receiver})"


class MessageBus:
    """
    消息总线

    支持：
    - 直接消息（点对点）
    - 广播消息（一对多）
    - 消息队列（异步传递）
    """

    def __init__(self):
        self._queues: dict[str, asyncio.Queue] = {}
        self._message_log: list[Message] = []
        self._subscribers: dict[str, list[str]] = {}  # agent_id → [subscribed_topics]

    def register_agent(self, agent_id: str) -> None:
        """注册 Agent"""
        if agent_id not in self._queues:
            self._queues[agent_id] = asyncio.Queue()
            logger.info(f"Agent registered on message bus: {agent_id}")

    async def send(self, message: Message) -> None:
        """发送消息"""
        self._message_log.append(message)

        if message.message_type == "broadcast":
            # 广播给所有注册的 Agent
            for agent_id, queue in self._queues.items():
                if agent_id != message.sender:
                    await queue.put(message)
            logger.info(f"Broadcast from {message.sender}: {message.content[:50]}...")
        else:
            # 直接消息
            queue = self._queues.get(message.receiver)
            if queue:
                await queue.put(message)
                logger.info(f"Message {message.sender} → {message.receiver}: {message.content[:50]}...")
            else:
                logger.warning(f"Receiver not found: {message.receiver}")

    async def receive(self, agent_id: str, timeout: float = 5.0) -> Optional[Message]:
        """接收消息"""
        queue = self._queues.get(agent_id)
        if not queue:
            return None
        try:
            return await asyncio.wait_for(queue.get(), timeout=timeout)
        except asyncio.TimeoutError:
            return None

    def get_history(self, agent_id: str = None, limit: int = 50) -> list[Message]:
        """获取消息历史"""
        messages = self._message_log
        if agent_id:
            messages = [
                m for m in messages
                if m.sender == agent_id or m.receiver == agent_id or m.message_type == "broadcast"
            ]
        return messages[-limit:]

    @property
    def message_count(self) -> int:
        return len(self._message_log)
```

### 3. 任务板 `team/task_board.py`

```python
"""
team/task_board.py —— 共享任务板

参考 Claude Code 的 TaskList / task list directory：
- 团队成员共享的任务列表
- 任务有状态、负责人、依赖关系
- 成员自动认领任务
"""

import asyncio
import logging
from dataclasses import dataclass, field
from enum import Enum
from typing import Optional

logger = logging.getLogger(__name__)


class TaskStatus(Enum):
    PENDING = "pending"
    IN_PROGRESS = "in_progress"
    COMPLETED = "completed"
    BLOCKED = "blocked"
    FAILED = "failed"


@dataclass
class Task:
    """任务（参考 Claude Code 的 task list entries）"""
    id: int
    description: str
    assigned_to: Optional[str] = None  # 负责人 agent_id
    status: TaskStatus = TaskStatus.PENDING
    priority: int = 1  # 1 = highest
    dependencies: list[int] = field(default_factory=list)
    result: Optional[str] = None
    created_by: str = ""

    @property
    def is_blocked(self) -> bool:
        """检查是否被阻塞"""
        return self.status == TaskStatus.BLOCKED

    @property
    def is_ready(self) -> bool:
        """是否可以开始（所有依赖已完成）"""
        return self.status == TaskStatus.PENDING and not self.is_blocked


class TaskBoard:
    """
    共享任务板

    参考 Claude Code 的团队任务列表：
    - 所有团队成员共享
    - 支持任务创建、分配、更新
    - 自动检测阻塞
    """

    def __init__(self):
        self._tasks: dict[int, Task] = {}
        self._next_id = 1
        self._lock = asyncio.Lock()

    async def create_task(
        self,
        description: str,
        priority: int = 1,
        dependencies: list[int] = None,
        created_by: str = "",
    ) -> Task:
        """创建任务"""
        async with self._lock:
            task = Task(
                id=self._next_id,
                description=description,
                priority=priority,
                dependencies=dependencies or [],
                created_by=created_by,
            )
            self._tasks[task.id] = task
            self._next_id += 1
            logger.info(f"Task created: #{task.id} - {description[:50]}...")
            return task

    async def claim_task(self, agent_id: str) -> Optional[Task]:
        """
        认领任务

        参考 Claude Code 的任务认领：
        - 选择优先级最高的就绪任务
        - 优先选择 ID 小的（依赖顺序）
        """
        async with self._lock:
            # 找就绪任务，按优先级和 ID 排序
            ready_tasks = [
                t for t in self._tasks.values()
                if t.is_ready and t.assigned_to is None
            ]

            # 检查依赖
            for task in sorted(ready_tasks, key=lambda t: (t.priority, t.id)):
                deps_met = all(
                    self._tasks.get(dep_id, Task(0, "")).status == TaskStatus.COMPLETED
                    for dep_id in task.dependencies
                )
                if deps_met:
                    task.assigned_to = agent_id
                    task.status = TaskStatus.IN_PROGRESS
                    logger.info(f"Task #{task.id} claimed by {agent_id}")
                    return task

            return None

    async def complete_task(self, task_id: int, result: str) -> bool:
        """完成任务"""
        async with self._lock:
            task = self._tasks.get(task_id)
            if not task:
                return False
            task.status = TaskStatus.COMPLETED
            task.result = result
            logger.info(f"Task #{task_id} completed by {task.assigned_to}")

            # 检查是否有被阻塞的任务现在可以执行了
            self._update_blocked_tasks()
            return True

    async def fail_task(self, task_id: int, error: str) -> bool:
        """标记任务失败"""
        async with self._lock:
            task = self._tasks.get(task_id)
            if not task:
                return False
            task.status = TaskStatus.FAILED
            task.result = f"Error: {error}"
            return True

    def _update_blocked_tasks(self) -> None:
        """更新被阻塞的任务状态"""
        for task in self._tasks.values():
            if task.status == TaskStatus.BLOCKED:
                deps_met = all(
                    self._tasks.get(dep_id, Task(0, "")).status == TaskStatus.COMPLETED
                    for dep_id in task.dependencies
                )
                if deps_met:
                    task.status = TaskStatus.PENDING

    def get_status(self) -> list[dict]:
        """获取所有任务状态"""
        return [
            {
                "id": t.id,
                "description": t.description,
                "assigned_to": t.assigned_to,
                "status": t.status.value,
                "priority": t.priority,
                "dependencies": t.dependencies,
            }
            for t in sorted(self._tasks.values(), key=lambda x: (x.priority, x.id))
        ]

    @property
    def pending_count(self) -> int:
        return sum(1 for t in self._tasks.values() if t.status == TaskStatus.PENDING)

    @property
    def completed_count(self) -> int:
        return sum(1 for t in self._tasks.values() if t.status == TaskStatus.COMPLETED)

    @property
    def total_count(self) -> int:
        return len(self._tasks)
```

### 4. 协调器 `team/coordinator.py`

```python
"""
team/coordinator.py —— 多智能体协调器

参考 Claude Code 的：
- TeamCreateTool（创建团队）
- TeamFile（团队配置）
- 主 Agent + 子 Agent 架构
- 任务分配和协调
"""

import asyncio
import json
import logging
import time
from dataclasses import dataclass, field
from typing import Optional

from .roles import BaseAgent, AgentRole, RoleType, ALL_ROLES, SimulatedAgent
from .message_bus import MessageBus, Message
from .task_board import TaskBoard, TaskStatus

logger = logging.getLogger(__name__)


@dataclass
class TeamConfig:
    """
    团队配置（参考 Claude Code 的 TeamFile）
    """
    name: str
    description: str = ""
    members: list[dict] = field(default_factory=list)
    created_at: float = field(default_factory=time.time)


class Coordinator:
    """
    多智能体协调器（主 Agent）

    参考 Claude Code 的 Team Lead Agent：
    1. 创建团队和分配角色
    2. 分解任务并分配到任务板
    3. 协调 Agent 执行
    4. 收集结果并整合
    """

    def __init__(self, team_name: str = "default-team"):
        self.team_name = team_name
        self.config = TeamConfig(name=team_name)
        self.agents: dict[str, BaseAgent] = {}
        self.message_bus = MessageBus()
        self.task_board = TaskBoard()
        self._running = False

    # ─── 团队管理 ───

    def add_role(self, role: AgentRole) -> BaseAgent:
        """
        添加角色到团队

        参考 Claude Code 的 spawn teammates via Agent tool
        """
        agent = SimulatedAgent(role)
        self.agents[role.agent_id] = agent
        self.message_bus.register_agent(role.agent_id)
        self.config.members.append({
            "agent_id": role.agent_id,
            "name": role.name,
            "role": role.role_type.value,
        })
        logger.info(f"Added team member: {role.name} ({role.role_type.value})")
        return agent

    def add_roles(self, role_types: list[RoleType]) -> list[BaseAgent]:
        """批量添加角色"""
        agents = []
        for rt in role_types:
            if rt in ALL_ROLES:
                agents.append(self.add_role(ALL_ROLES[rt]))
        return agents

    # ─── 任务编排 ───

    async def plan_and_execute(self, project_description: str) -> dict:
        """
        规划并执行项目

        完整的多 Agent 协作流程：
        1. Planner 分解任务
        2. 创建任务到任务板
        3. Agent 认领并执行
        4. 收集结果
        """
        logger.info(f"Starting project: {project_description}")
        self._running = True

        # Step 1: Planner 分解任务
        planner = self.agents.get("planner")
        if not planner:
            # 如果没有 Planner，使用默认任务分解
            tasks = self._default_decomposition(project_description)
        else:
            plan_result = await planner.execute(project_description)
            tasks = self._parse_plan(plan_result)

        # Step 2: 创建任务到任务板
        task_ids = []
        for task_info in tasks:
            task = await self.task_board.create_task(
                description=task_info["description"],
                priority=task_info.get("priority", 1),
                dependencies=task_info.get("dependencies", []),
                created_by="coordinator",
            )
            task_ids.append(task.id)

        # Step 3: Agent 认领并执行
        results = await self._execute_tasks()

        # Step 4: 整合结果
        summary = await self._summarize_results(results)

        self._running = False
        return {
            "project": project_description,
            "tasks_completed": self.task_board.completed_count,
            "tasks_total": self.task_board.total_count,
            "results": results,
            "summary": summary,
        }

    def _default_decomposition(self, description: str) -> list[dict]:
        """默认任务分解"""
        return [
            {
                "id": 1,
                "description": f"Research: {description}",
                "assigned_role": "researcher",
                "priority": 1,
                "dependencies": [],
            },
            {
                "id": 2,
                "description": f"Implement: {description}",
                "assigned_role": "coder",
                "priority": 1,
                "dependencies": [1],
            },
            {
                "id": 3,
                "description": f"Review code for: {description}",
                "assigned_role": "reviewer",
                "priority": 2,
                "dependencies": [2],
            },
            {
                "id": 4,
                "description": f"Test: {description}",
                "assigned_role": "tester",
                "priority": 2,
                "dependencies": [2],
            },
            {
                "id": 5,
                "description": f"Document: {description}",
                "assigned_role": "writer",
                "priority": 3,
                "dependencies": [2],
            },
        ]

    def _parse_plan(self, plan_result: str) -> list[dict]:
        """解析 Planner 的输出"""
        # 简化实现：返回默认分解
        return self._default_decomposition(plan_result)

    async def _execute_tasks(self) -> dict[int, str]:
        """
        执行任务板上的所有任务

        参考 Claude Code 的团队工作流：
        - Agent 自动认领任务
        - 完成后检查下一个
        """
        results = {}
        max_iterations = 20  # 防止无限循环
        iteration = 0

        while self.task_board.pending_count > 0 and iteration < max_iterations:
            iteration += 1
            progress_made = False

            # 每个空闲 Agent 尝试认领任务
            for agent_id, agent in self.agents.items():
                if agent.is_busy:
                    continue

                task = await self.task_board.claim_task(agent_id)
                if task:
                    progress_made = True
                    try:
                        # 收集依赖任务的输出作为上下文
                        context = self._gather_context(task.dependencies)
                        result = await agent.execute(task.description, context)
                        await self.task_board.complete_task(task.id, result)
                        results[task.id] = result

                        # 发送完成通知
                        await self.message_bus.send(Message(
                            sender=agent_id,
                            receiver="coordinator",
                            content=f"Task #{task.id} completed: {task.description[:50]}...",
                        ))
                    except Exception as e:
                        await self.task_board.fail_task(task.id, str(e))
                        results[task.id] = f"Error: {e}"

            if not progress_made:
                # 检查是否有阻塞的任务
                blocked = [
                    t for t in self.task_board.get_status()
                    if t["status"] == "blocked" or (
                        t["status"] == "pending" and t["assigned_to"] is None
                    )
                ]
                if not blocked or all(
                    t["status"] == "pending" and t["dependencies"]
                    for t in blocked
                ):
                    # 所有 pending 任务都有未满足的依赖，等待
                    await asyncio.sleep(0.1)
                else:
                    break

        return results

    def _gather_context(self, dependency_ids: list[int]) -> dict:
        """收集依赖任务的输出"""
        context = {}
        for dep_id in dependency_ids:
            # 从任务板获取结果（简化实现）
            context[f"task_{dep_id}"] = f"Result from task {dep_id}"
        return context

    async def _summarize_results(self, results: dict[int, str]) -> str:
        """整合所有结果"""
        parts = [f"## Project Summary: {self.team_name}\n"]

        for task_id, result in sorted(results.items()):
            parts.append(f"\n### Task #{task_id}")
            # 截取结果的前 200 字符
            preview = result[:200] + "..." if len(result) > 200 else result
            parts.append(preview)

        parts.append(f"\n---\n**Total tasks completed: {len(results)}**")
        return "\n".join(parts)

    # ─── 状态查询 ───

    def get_team_status(self) -> dict:
        """获取团队状态"""
        return {
            "team_name": self.team_name,
            "members": [
                {
                    "name": agent.role.name,
                    "role": agent.role.role_type.value,
                    "status": "busy" if agent.is_busy else "idle",
                    "current_task": agent.current_task,
                }
                for agent in self.agents.values()
            ],
            "tasks": self.task_board.get_status(),
            "messages": self.message_bus.message_count,
        }
```

```python
        t2 = await coord.task_board.create_task(
            "Implement the solution",
            dependencies=[t1.id],
        )
        t3 = await coord.task_board.create_task(
            "Review the code",
            dependencies=[t2.id],
        )

        # 执行
        results = await coord._execute_tasks()

        # 所有任务应该完成
        assert len(results) == 3
        assert coord.task_board.completed_count == 3

        # 检查执行顺序（通过结果中的内容）
        assert 1 in results  # Research
        assert 2 in results  # Implement
        assert 3 in results  # Review

    @pytest.mark.asyncio
    async def test_parallel_independent_tasks(self):
        """测试并行独立任务"""
        coord = Coordinator("parallel")

        # 添加 3 个 Worker
        for i in range(3):
            worker = AgentRole(
                agent_id=f"worker-{i}",
                role_type=RoleType.CODER,
                name=f"Worker-{i}",
                system_prompt="You are a coding worker.",
            )
            coord.add_role(worker)

        # 创建 3 个独立任务
        for desc in ["Task A", "Task B", "Task C"]:
            await coord.task_board.create_task(desc, priority=1)

        results = await coord._execute_tasks()

        assert len(results) == 3
        assert coord.task_board.completed_count == 3
```

## 📝 本章小结

本章我们深入学习了 Claude Code 的多智能体协作机制，并实现了完整的 Team Mode 系统：

| 概念 | Claude Code 实现 | 我们的 Python 实现 |
|------|-----------------|-------------------|
| 团队创建 | TeamCreateTool | Coordinator + TeamConfig |
| 团队文件 | TeamFile (JSON) | TeamConfig (dataclass) |
| 角色分配 | AgentType + AgentDefinition | AgentRole + RoleType |
| 任务列表 | ~/.claude/tasks/{team}/ | TaskBoard |
| 消息传递 | SendMessage + 通知系统 | MessageBus |
| 任务认领 | TaskUpdate with owner | claim_task |
| 空闲通知 | 自动 idle notification | is_busy 标志 |
| 依赖管理 | Task dependencies | Task.dependencies |

**三种协作模式：**

1. **流水线模式**：Planner → Researcher → Coder → Reviewer → Tester，适合复杂项目
2. **并行模式**：多个 Worker 同时处理独立任务，适合批量工作
3. **主从模式**：Lead 分配任务，Worker 执行并汇报，Claude Code 的标准模式

**核心收获：**

1. **多 Agent 的核心是协调**——没有好的协调器，多个 Agent 只会互相干扰
2. **任务板是协作的枢纽**——所有 Agent 通过任务板发现和认领工作
3. **消息总线是神经系统**——Agent 间通过消息传递信息和状态
4. **依赖管理决定执行顺序**——正确处理依赖是避免混乱的关键
5. **空闲是正常状态**——Agent 完成任务后空闲，等待新任务，不要视为错误

### 练习 1：实现 Agent 自动扩缩容

> **提示**：当任务板上有大量 pending 任务时，自动创建新的 Worker Agent。当所有 Agent 空闲超过一段时间后，自动释放。参考云服务的 auto-scaling 概念。

### 练习 2：实现 Agent 记忆共享

> **提示**：创建一个 SharedMemory 类，所有 Agent 可以读写共享的知识库。Agent A 的研究结果可以被 Agent B 直接使用。可以用向量数据库（如 ChromaDB）实现语义搜索。

### 练习 3：实现 Agent 冲突解决

> **提示**：当多个 Agent 同时修改同一文件时，检测冲突并解决。可以用文件锁或乐观并发控制。参考 Git 的合并策略。

### 练习 4：实现 Agent 绩效评估

> **提示**：跟踪每个 Agent 的任务完成率、耗时、质量评分。生成团队绩效报告，用于优化任务分配策略。

**答案提示：**
- 练习 1：监控 `pending_count / available_agents` 比率，超过阈值时 `add_role()`
- 练习 2：`SharedMemory` 类用 dict 存储，`put(key, value)` / `get(key)` / `search(query)`
- 练习 3：文件写入前检查 `file_locks`，冲突时用三路合并（three-way merge）
- 练习 4：在 Task 中添加 `quality_score` 字段，Agent 中添加 `stats` 计数器
