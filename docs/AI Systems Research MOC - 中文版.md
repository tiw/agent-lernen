### 关键模式速查

| 模式 | 最佳实现 | 适用场景 |
|------|----------|----------|
| **三角色分离** | Claude Code | 需要严格验证的任务 |
| **阶段编排** | oh-my-codex | 复杂多步骤工作流 |
| **上下文压缩** | Hermes | 长会话管理 |
| **多平台网关** | OpenClaw | 跨平台部署 |
| **子代理控制** | OpenClaw | 大规模代理编排 |

### 研究方法论

基于 **Harness工程统一框架**:

1. **4D可扩展性模型**
   - 时间维度: 时间维度（长期一致性）
   - 空间维度: 空间维度（并行协调）
   - 交互维度: 交互维度（人机协作）
   - 知识维度: 知识维度（信息熵管理）

2. **递归规划架构**
   - Planner → Generator → Evaluator 分离
   - 状态隔离机制
   - 自评失真预防

3. **控制论映射**
   - 反馈系统
   - 控制机制
   - 通信基础设施
   - 熵管理
