## 📖 核心概念

### Agent 项目的通用架构

```
用户任务描述
    │
    ▼
┌─────────────────┐
│  任务解析器      │  ← 理解任务类型，选择策略
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  工具编排器      │  ← 按顺序/并行调用工具
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  结果处理器      │  ← 格式化、验证、输出
└────────┬────────┘
         │
         ▼
    最终结果
```

### 三种项目模式

| 模式 | 特点 | 适用场景 |
|------|------|---------|
| 分析型 | 读取 → 分析 → 报告 | 代码审查、日志分析 |
| 生成型 | 理解 → 规划 → 生成 | 文档生成、代码生成 |
| 交互型 | 询问 → 执行 → 反馈 | 数据分析、调试助手 |

## 💻 三个实战项目

### 项目 1：自动代码审查助手

**功能：**
- 扫描指定目录的代码文件
- 使用 LLM 分析代码质量
- 生成结构化的审查报告
- 支持自定义审查规则

**架构组件：**
- `CodeScanner` - 代码文件扫描器
- `StaticAnalyzer` - 静态代码分析器（规则引擎）
- `LLMReviewer` - 基于 LLM 的深度审查
- `CodeReviewAgent` - 代码审查 Agent

**审查类别：**
- 🔴 Critical（严重）- 必须修复
- 🟡 Warning（警告）- 建议修复
- ℹ️ Info（信息）- 可选优化
- ✅ Positive（正面）- 做得好的地方

**静态检查规则（Python）：**
- 过长的行（>120 字符）
- 硬编码的密钥模式
- bare except
- print 调试代码
- TODO/FIXME 注释
- 通配符导入
- 文件过长

### 项目 2：智能文档生成器

**功能：**
- 分析代码仓库结构
- 自动生成 README、API 文档、架构文档
- 支持多种输出格式（Markdown、HTML、PDF）
- 可自定义文档模板

**架构组件：**
- `DocTemplates` - 文档模板集合
- `CodebaseAnalyzer` - 代码仓库分析器
- `DocGenerator` - 文档生成器

**生成文档类型：**
- README.md - 项目说明
- API.md - API 文档
- ARCHITECTURE.md - 架构文档

### 项目 3：数据分析 Agent

**功能：**
- 读取 CSV/JSON/Excel 数据
- 自动数据探索（描述性统计、分布分析）
- 自然语言查询数据
- 生成可视化图表
- 输出分析报告

**架构组件：**
- `DataTools` - 数据分析工具集
- `Visualizer` - 数据可视化工具
- `DataAnalystAgent` - 数据分析 Agent

**数据工具：**
- `load_csv()` - 加载 CSV 文件
- `load_json()` - 加载 JSON 文件
- `describe()` - 描述性统计
- `query()` - 简单数据查询

**可视化：**
- ASCII 条形图（无需 matplotlib）
- 汇总表格

## 🚀 部署与运维

### 1. 容器化部署

**Dockerfile 要点：**
- 基于 python:3.12-slim
- 安装依赖
- 创建数据目录
- 运行 CLI 接口

**docker-compose 配置：**
- 挂载数据卷
- 环境变量配置
- 交互式终端

### 2. 配置管理

**AgentConfig 数据类：**
- LLM 配置（provider, model, api_key, base_url）
- 安全配置（strict_mode, sandbox_enabled）
- Hook 配置
- 日志配置

**环境变量加载：**
- `from_env()` 类方法
- 默认值设置

### 3. 监控与告警

**Monitor 监控器：**
- 记录请求次数
- 记录错误计数
- 记录响应时间
- 健康状态检查

**健康状态：**
- healthy（错误率 < 10%）
- degraded（错误率 10%-50%）
- unhealthy（错误率 > 50%）

### 4. 运维检查清单

| 检查项 | 频率 | 方法 |
|--------|------|------|
| API 密钥有效期 | 每月 | 检查环境变量 |
| 磁盘空间 | 每日 | `df -h` |
| 审计日志大小 | 每周 | 清理过期日志 |
| 会话持久化文件 | 每周 | 清理过期会话 |
| 依赖更新 | 每月 | `pip list --outdated` |
| 安全策略更新 | 每月 | 审查白名单/黑名单 |
